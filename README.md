# Legends Unbound

**2D Top-Down Action-RPG** з варіативною бойовою системою (меч + лук), інвентарем, прокачкою та системою збереження.

## 🎮 Про гру

«Legends Unbound» — це функціональний прототип 2D Action-RPG з верхнім ракурсом (top-down).  
Гра демонструє сучасні підходи до розробки ігор з акцентом на **чисту архітектуру**, принципи ООП та data-driven design.

### Основні можливості

- **Варіативна бойова система** — перемикання між ближнім боєм (меч) та дальнім боєм (лук)
- **Система інвентарю** з екіпіруванням зброї та використанням їжі для відновлення здоров’я
- **Прокачка персонажа** — рівень, досвід, характеристики (сила, спритність), skill points
- **Штучний інтелект ворогів** на базі NavMesh (Roaming → Chasing → Attacking)
- **Система збереження/завантаження** прогресу через JSON
- **Зручний UI/UX** — HUD, меню персонажа, інвентар, меню паузи
- **Data-driven дизайн** — зброя, вороги та предмети завантажуються з JSON-файлів

## Технології та інструменти

- **Unity 6**
- **C#**
- **Input System**
- **NavMesh + NavMeshAgent**
- **Animator + 2D Animation**
- **uGUI (Canvas)**
- **JsonUtility** (серiалізація)

## Як запустити

1. Клонуй репозиторій на свій комп’ютер:git clone https://github.com/Stitlex/Legends-Unbound.git
2. Відкрий проєкт у Unity:
3. Запусти Unity Hub
4. Натисни Add (Додати проєкт)
5. Обери папку Legends-Unbound, яку ти щойно склоновував
5. Переконайся, що вибрана версія Unity 6.0 або новіша
6. Натисни Open

Налаштуй проєкт (перший запуск):
1. Після відкриття проєкту Unity може запропонувати імпортувати пакети або оновити індекси — дочекайся завершення.
2. Перейди в папку Assets/Scenes/
3. Відкрий сцену MainMenu (рекомендовано для початку)

Запусти гру:
Натисни кнопку Play у верхній панелі Unity Editor

Альтернативний спосіб (якщо хочеш відразу в гру):
1. Відкрий сцену GameScene (або як вона у тебе називається)
2. Натисни Play
